# ToneForge AI

AI-powered email assistant: formalize, translate, reply, negotiate, and parse legal emails.

## Features
- **Formalize**: Elevate casual drafts into professional communication.
- **Smart Reply**: Generate intelligent, context-aware responses.
- **Negotiation**: Strategic simulation for complex discussions.
- **Legal Parser**: Deconstruct risks and obligations in legal text.

## Tech Stack
- **Backend**: FastAPI
- **LLM**: Groq (Llama 3 / Mixtral via LangChain)
- **Frontend**: Tailwind CSS + Lucide Icons
